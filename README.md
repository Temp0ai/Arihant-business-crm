# ☕ Tea & Coffee CRM

Android CRM app for tea and coffee businesses

## Features
- 📧 Gmail lead extraction
- 🤖 Gemini AI follow-ups
- 💬 Bulk WhatsApp messaging
- 📸 AI image generation
- 🔍 SEO tools
- 📊 Analytics dashboard

## Setup
1. Clone repo
2. Add secrets.properties with your API keys
3. Add google-services.json
4. Build in Android Studio

## API Keys Required
- Gemini: https://aistudio.google.com (Free)
- Gmail OAuth: https://console.cloud.google.com (Free)

## Tech Stack
- Kotlin + Jetpack Compose
- Gemini AI (Flash + Pro)
- Room Database
- Hilt DI
- Retrofit
